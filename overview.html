Overview¶

Icepack uses the library firedrake for finite element analysis. The important objects from firedrake that you’ll be using are:

Mesh()
Create the geometry for the domain
FunctionSpace()
Create a finite element basis over a given mesh for representing fields
Function
Represents a scalar, vector, or tensor field defined on some function space

Icepack consists of routines that solve for various quantities encountered in ice sheet physics. For example, the class IceShelf has methods diagnostic_solve() and prognostic_solve() that compute (respectively) the ice velocity and thickness. The computed velocity and thickness are firedrake.Function objects that you can plot or analyze using the various routines in firedrake.

For a thorough reference on ice sheet physics, see [1] and [2]. For an introduction to the finite element method, see [3] and [4].

[1]Ralf Greve and Heinz Blatter. Dynamics of ice sheets and glaciers. Springer Science & Business Media, 2009.
[2]Kurt M Cuffey and William Stanley Bryce Paterson. The physics of glaciers. Academic Press, 2010.
[3]Mark S Gockenbach. Understanding and implementing the finite element method. Volume 97. SIAM, 2006.
[4]Dietrich Braess. Finite elements: Theory, fast solvers, and applications in solid mechanics. Cambridge University Press, 2007.
