Bases: object

Class for modelling the flow of grounded ice This class provides functions that solve for the velocity, thickness, and surface elevation of a grounded area of slow flowing ice.

icepack.models.shallow_ice.gravity(**kwargs)[source]¶

Return the gravity term for the shallow ice action functional

The gravity function for the shallow ice action functional is

\[E(u) = \int_\Omega\frac{2A(\varrho_I g)**n}{n+2} (\nabla h^2\cdot u) h^{n+1} \nabla s^{n-1}\; dx\]
