icepack.models.ice_shelf module¶

class icepack.models.ice_shelf.IceShelf(viscosity=<function viscosity_depth_averaged>, side_friction=<function side_friction>, gravity=<function gravity>, terminus=<function terminus>, penalty=<function normal_flow_penalty>)[source]¶

Bases: object

Class for modelling the flow of floating ice shelves

This class provides functions that solve for the velocity and thickness of a floating ice shelf. The relevant physics can be found in ch. 6 of Greve and Blatter.

See also

icepack.models.viscosity.viscosity_depth_averaged()

action(u, h, **kwargs)[source]¶

Return the action functional that gives the ice shelf diagnostic model as the Euler-Lagrange equations

The action functional for the ice shelf diagnostic model is

\[E(u) = \int_\Omega\left(\frac{n}{n + 1}hM:\dot\varepsilon - \frac{1}{2}\varrho gh^2\nabla\cdot u\right)dx\]

where \(u\) is the velocity, \(h\) is the ice thickness, \(\dot\varepsilon\) is the strain-rate tensor, and \(M\) is the membrane stress tensor.

Parameters:
  • u (firedrake.Function) – ice velocity
  • h (firedrake.Function) – ice thickness
Returns:

E – the ice shelf action functional
Return type:

firedrake.Form
Other Parameters:
 

**kwargs – All other keyword arguments will be passed on to the viscosity and gravity functionals. The ice fluidity coefficient, for example, is passed as a keyword argument.

quadrature_degree(u, h, **kwargs)[source]¶

Return the quadrature degree necessary to integrate the action functional accurately

Firedrake uses a very conservative algorithm for estimating the number of quadrature points necessary to integrate a given expression. By exploiting known structure of the problem, we can reduce the number of quadrature points while preserving accuracy.

scale(u, h, **kwargs)[source]¶

Return the positive, convex part of the action functional

The positive part of the action functional is used as a dimensional scale to determine when to terminate an optimization algorithm.

icepack.models.ice_shelf.terminus(u, h, ice_front_ids=())[source]¶

Return the terminus stress part of the ice shelf action functional

The power exerted due to stress at the calving terminus \(\Gamma\) is

\[E(u) = \int_\Gamma\varrho gh^2u\cdot\nu\hspace{2pt}ds\]

We assume that sea level is at \(z = 0\) for calculating the water depth.
